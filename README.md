# Sentiment Analysis Chatbox

This is an AI-powered chatbox that uses Natural Language Processing (NLP) to detect the sentiment of user inputs. It is built with Python, Flask, and TensorFlow.

## Features
- **Real-time Sentiment Analysis:** Classifies text as Positive, Negative, or Neutral.
- **Web Interface:** Simple and clean UI using HTML/CSS.
- **Custom Model:** Trained on a balanced dataset using TensorFlow/Keras.
